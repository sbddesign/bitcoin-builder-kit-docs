# Icon Usage

Learn how to properly implement and use icons in your applications with accessibility and performance in mind.

## Accessibility

Icons should always be accessible to screen readers and other assistive technologies.

### JavaScript (Web Components)

```javascript
// Accessible icon component
class AccessibleIcon extends HTMLElement {
  static get observedAttributes() {
    return ['name', 'label', 'decorative'];
  }

  connectedCallback() {
    this.render();
  }

  render() {
    const name = this.getAttribute('name');
    const label = this.getAttribute('label');
    const decorative = this.getAttribute('decorative') === 'true';
    
    const svg = document.createElementNS('http://www.w3.org/2000/svg', 'svg');
    svg.setAttribute('width', '24');
    svg.setAttribute('height', '24');
    svg.setAttribute('viewBox', '0 0 24 24');
    svg.setAttribute('fill', 'currentColor');
    
    if (decorative) {
      svg.setAttribute('aria-hidden', 'true');
    } else {
      svg.setAttribute('role', 'img');
      svg.setAttribute('aria-label', label || name);
    }
    
    svg.innerHTML = this.getIconPath(name);
    this.innerHTML = '';
    this.appendChild(svg);
  }

  getIconPath(name) {
    // Icon path data...
    return '<path d="..."/>';
  }
}

customElements.define('accessible-icon', AccessibleIcon);

// Usage examples
const decorativeIcon = document.createElement('accessible-icon');
decorativeIcon.setAttribute('name', 'star');
decorativeIcon.setAttribute('decorative', 'true');

const interactiveIcon = document.createElement('accessible-icon');
interactiveIcon.setAttribute('name', 'close');
interactiveIcon.setAttribute('label', 'Close dialog');
```

### React

```tsx
import { Icon } from '@your-package/icons';

interface AccessibleIconProps {
  name: string;
  label?: string;
  decorative?: boolean;
  size?: number | string;
  color?: string;
}

function AccessibleIcon({ 
  name, 
  label, 
  decorative = false, 
  size = 24, 
  color = 'currentColor' 
}: AccessibleIconProps) {
  const svgProps = {
    width: size,
    height: size,
    viewBox: '0 0 24 24',
    fill: color,
    ...(decorative 
      ? { 'aria-hidden': true }
      : { role: 'img', 'aria-label': label || name }
    )
  };

  return (
    <svg {...svgProps}>
      <path d={getIconPath(name)} />
    </svg>
  );
}

// Usage examples
function ButtonWithIcon({ children, onClick }) {
  return (
    <button onClick={onClick} style={{ display: 'flex', alignItems: 'center', gap: '0.5rem' }}>
      <AccessibleIcon name="plus" label="Add item" />
      {children}
    </button>
  );
}

function DecorativeIcon() {
  return (
    <div style={{ display: 'flex', alignItems: 'center', gap: '0.5rem' }}>
      <AccessibleIcon name="star" decorative />
      <span>Featured content</span>
    </div>
  );
}
```

## Performance

Optimize icon loading and rendering for better performance.

### Lazy Loading

```tsx
import { lazy, Suspense } from 'react';

// Lazy load icon components
const IconLibrary = lazy(() => import('@your-package/icons/library'));

function App() {
  return (
    <Suspense fallback={<div>Loading icons...</div>}>
      <IconLibrary />
    </Suspense>
  );
}
```

```javascript
customElements.define('icon-sprite', IconSprite);

// Use icons from sprite
function useIconFromSprite(name) {
  return `
    <svg width="24" height="24" viewBox="0 0 24 24" fill="currentColor">
      <use href="#${name}"></use>
    </svg>
  `;
}
```

## Best Practices

1. **Use semantic names** - Choose descriptive names that indicate purpose
2. **Consistent sizing** - Stick to standard sizes (16px, 20px, 24px, 32px)
3. **Proper contrast** - Ensure icons meet accessibility contrast requirements
4. **Loading states** - Provide loading indicators for dynamic icons
5. **Fallbacks** - Always provide fallback text for critical icons
