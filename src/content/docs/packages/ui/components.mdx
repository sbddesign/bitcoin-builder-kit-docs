# Components

This section documents all available UI components in the package.

## Button

A versatile button component with multiple variants.

```tsx
import { Button } from '@your-package/ui'

<Button variant="primary">Click me</Button>
```

## Card

A container component for grouping related content.

```tsx
import { Card } from '@your-package/ui'

<Card>
  <h3>Card Title</h3>
  <p>Card content goes here</p>
</Card>
```

## Input

Form input components with validation support.

```tsx
import { Input } from '@your-package/ui'

<Input 
  type="text" 
  placeholder="Enter your name"
  required 
/>
```
