# Color Tokens

Our color system provides a comprehensive palette of colors with semantic meaning and consistent usage across all components.

## Primary Colors

The primary color palette forms the foundation of our design system.

### JavaScript (Web Components)

```javascript
// Using CSS custom properties for colors
const primaryButton = document.createElement('button');
primaryButton.style.setProperty('--bg-color', 'var(--token-color-primary-500)');
primaryButton.style.setProperty('--text-color', 'var(--token-color-primary-contrast)');
primaryButton.style.setProperty('--hover-color', 'var(--token-color-primary-600)');

// Apply hover effects
primaryButton.addEventListener('mouseenter', () => {
  primaryButton.style.backgroundColor = 'var(--token-color-primary-600)';
});
```

### React

```tsx
import { tokens } from '@your-package/tokens';

function PrimaryButton({ children, onClick }) {
  const buttonStyles = {
    backgroundColor: tokens.colors.primary[500],
    color: tokens.colors.primary.contrast,
    border: 'none',
    padding: `${tokens.spacing.sm} ${tokens.spacing.md}`,
    borderRadius: tokens.radius.md,
    cursor: 'pointer',
    transition: 'background-color 0.2s ease',
  };

  const handleMouseEnter = (e) => {
    e.target.style.backgroundColor = tokens.colors.primary[600];
  };

  return (
    <button 
      style={buttonStyles}
      onClick={onClick}
      onMouseEnter={handleMouseEnter}
    >
      {children}
    </button>
  );
}
```

## Semantic Colors

Semantic colors provide meaning and context to UI elements.

```css
/* CSS Custom Properties */
:root {
  --token-color-success: #10b981;
  --token-color-warning: #f59e0b;
  --token-color-error: #ef4444;
  --token-color-info: #3b82f6;
}
```

## Color Scale

Each color family includes multiple shades for different use cases.

| Token | Value | Usage |
|-------|-------|-------|
| `--token-color-primary-50` | `#eff6ff` | Light backgrounds |
| `--token-color-primary-500` | `#3b82f6` | Primary actions |
| `--token-color-primary-900` | `#1e3a8a` | Dark text |
