# Configuration

Configure Bitcoin Tip Jar with various options to match your application's needs.

## Basic Settings

### JavaScript (Web Components)

```javascript
// Create configured tip jar
const tipJar = document.createElement('bitcoin-tip-jar');

// Required settings
tipJar.setAttribute('address', 'bc1qxy2kgdygjrsqtzq2n0yrf2493p83kkfjhx0wlh');

// Optional settings
tipJar.setAttribute('amount', '0.0001'); // Default tip amount
tipJar.setAttribute('currency', 'btc'); // Display currency
tipJar.setAttribute('theme', 'auto'); // Theme: light, dark, auto
tipJar.setAttribute('size', 'medium'); // Size: small, medium, large
tipJar.setAttribute('position', 'bottom-right'); // Position: top-left, top-right, bottom-left, bottom-right

// Event listeners
tipJar.addEventListener('tip-initiated', (event) => {
  console.log('Tip initiated:', event.detail);
});

tipJar.addEventListener('tip-completed', (event) => {
  console.log('Tip completed:', event.detail);
});

document.body.appendChild(tipJar);
```

### React

```tsx
import { BitcoinTipJar } from '@bitcoin-tip-jar/react';
import { useState } from 'react';

function ConfiguredTipJar() {
  const [tipAmount, setTipAmount] = useState(0.0001);
  const [theme, setTheme] = useState('auto');

  const handleTipInitiated = (event) => {
    console.log('Tip initiated:', event.detail);
  };

  const handleTipCompleted = (event) => {
    console.log('Tip completed:', event.detail);
  };

  return (
    <div>
      <BitcoinTipJar
        address="bc1qxy2kgdygjrsqtzq2n0yrf2493p83kkfjhx0wlh"
        amount={tipAmount}
        currency="btc"
        theme={theme}
        size="medium"
        position="bottom-right"
        onTipInitiated={handleTipInitiated}
        onTipCompleted={handleTipCompleted}
      />
      
      <div style={{ marginTop: '1rem' }}>
        <label>
          Tip Amount (BTC):
          <input 
            type="number" 
            value={tipAmount} 
            onChange={(e) => setTipAmount(parseFloat(e.target.value))}
            step="0.0001"
            min="0.0001"
          />
        </label>
        
        <label>
          Theme:
          <select value={theme} onChange={(e) => setTheme(e.target.value)}>
            <option value="auto">Auto</option>
            <option value="light">Light</option>
            <option value="dark">Dark</option>
          </select>
        </label>
      </div>
    </div>
  );
}
```

## Advanced Configuration

### Multiple Amounts

```javascript
// Configure multiple tip amounts
const tipJar = document.createElement('bitcoin-tip-jar');
tipJar.setAttribute('address', 'your-bitcoin-address');
tipJar.setAttribute('amounts', '0.0001,0.0005,0.001,0.005'); // Comma-separated amounts
tipJar.setAttribute('currency', 'btc');
```

### Custom QR Code

```tsx
import { BitcoinTipJar } from '@bitcoin-tip-jar/react';

function CustomQRTipJar() {
  return (
    <BitcoinTipJar
      address="bc1qxy2kgdygjrsqtzq2n0yrf2493p83kkfjhx0wlh"
      amount={0.0001}
      qrCode={{
        size: 200,
        color: '#000000',
        backgroundColor: '#ffffff',
        logo: '/path/to/logo.png'
      }}
    />
  );
}
```

## Environment Variables

```javascript
// Use environment variables for sensitive data
const tipJar = document.createElement('bitcoin-tip-jar');
tipJar.setAttribute('address', process.env.BITCOIN_ADDRESS);
tipJar.setAttribute('amount', process.env.DEFAULT_TIP_AMOUNT || '0.0001');
tipJar.setAttribute('network', process.env.BITCOIN_NETWORK || 'mainnet');
```
