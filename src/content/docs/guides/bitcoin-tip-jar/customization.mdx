# Customization

Customize the appearance, styling, and behavior of Bitcoin Tip Jar to match your application's design.

## Styling

### JavaScript (Web Components)

```javascript
// Custom styling with CSS
const style = document.createElement('style');
style.textContent = `
  bitcoin-tip-jar {
    --tip-jar-primary-color: #f7931a;
    --tip-jar-background: #ffffff;
    --tip-jar-text-color: #000000;
    --tip-jar-border-radius: 12px;
    --tip-jar-shadow: 0 4px 12px rgba(0, 0, 0, 0.1);
    --tip-jar-font-family: 'Inter', sans-serif;
  }
  
  bitcoin-tip-jar::part(button) {
    background: linear-gradient(135deg, #f7931a, #ff6b35);
    border: none;
    border-radius: 8px;
    padding: 12px 24px;
    font-weight: 600;
    transition: all 0.3s ease;
  }
  
  bitcoin-tip-jar::part(button):hover {
    transform: translateY(-2px);
    box-shadow: 0 6px 20px rgba(247, 147, 26, 0.3);
  }
`;
document.head.appendChild(style);

// Create styled tip jar
const tipJar = document.createElement('bitcoin-tip-jar');
tipJar.setAttribute('address', 'bc1qxy2kgdygjrsqtzq2n0yrf2493p83kkfjhx0wlh');
tipJar.setAttribute('amount', '0.0001');
tipJar.setAttribute('class', 'custom-tip-jar');
```

### React

```tsx
import { BitcoinTipJar } from '@bitcoin-tip-jar/react';
import styled from 'styled-components';

const StyledTipJar = styled(BitcoinTipJar)`
  --tip-jar-primary-color: #f7931a;
  --tip-jar-background: #ffffff;
  --tip-jar-text-color: #000000;
  --tip-jar-border-radius: 12px;
  --tip-jar-shadow: 0 4px 12px rgba(0, 0, 0, 0.1);
  
  &::part(button) {
    background: linear-gradient(135deg, #f7931a, #ff6b35);
    border: none;
    border-radius: 8px;
    padding: 12px 24px;
    font-weight: 600;
    transition: all 0.3s ease;
    
    &:hover {
      transform: translateY(-2px);
      box-shadow: 0 6px 20px rgba(247, 147, 26, 0.3);
    }
  }
`;

function CustomStyledTipJar() {
  return (
    <StyledTipJar
      address="bc1qxy2kgdygjrsqtzq2n0yrf2493p83kkfjhx0wlh"
      amount={0.0001}
    />
  );
}
```

## Custom Themes

### Dark Theme

```css
/* Dark theme styles */
bitcoin-tip-jar[theme="dark"] {
  --tip-jar-primary-color: #f7931a;
  --tip-jar-background: #1a1a1a;
  --tip-jar-text-color: #ffffff;
  --tip-jar-border-color: #333333;
  --tip-jar-shadow: 0 4px 12px rgba(0, 0, 0, 0.3);
}
```

### Custom Brand Theme

```tsx
// Brand-specific theme
const brandTheme = {
  primaryColor: '#your-brand-color',
  backgroundColor: '#ffffff',
  textColor: '#your-text-color',
  borderRadius: '8px',
  fontFamily: 'Your Brand Font',
  buttonStyle: {
    background: 'linear-gradient(45deg, #brand-start, #brand-end)',
    border: 'none',
    borderRadius: '6px',
    padding: '10px 20px',
    fontWeight: '500'
  }
};

function BrandedTipJar() {
  return (
    <BitcoinTipJar
      address="bc1qxy2kgdygjrsqtzq2n0yrf2493p83kkfjhx0wlh"
      amount={0.0001}
      theme={brandTheme}
    />
  );
}
```
